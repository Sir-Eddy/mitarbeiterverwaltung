# Mitarbeiterverwaltung CLI

Ein einfaches CLI-Programm zur Verwaltung von Mitarbeitern, erstellt im Rahmen des VHB-Kurses **C++**. Nur unter Windows lauffähig.

## 📋 Funktionen

- **Mitarbeiter anlegen**: Erfassung von Daten und Urlaubsberechnung.  
- **Mitarbeiter löschen**: Entfernen von Einträgen.  
- **Urlaub buchen**: Aktualisierung des Resturlaubs.  
- **Resturlaub anzeigen**: Übersicht des verbleibenden Urlaubs.  
- **Alle Mitarbeiter anzeigen**: Auflistung gespeicherter Daten.  

## ⚙️ Voraussetzungen

- **C++ Compiler** (z. B. Visual Studio oder MinGW)  
- Windows-Betriebssystem  

## 📂 Verwendung

1. Kompilieren:  
   ```bash
   g++ -o mitarbeiterverwaltung.exe main.cpp
   ```
2. Starten:  
   ```bash
   mitarbeiterverwaltung.exe
   ```

## 💡 Hinweise

- Maximal 500 Mitarbeiter.  
- Eingegebene Daten werden nur zur Laufzeit gespeichert.
